# Sensing with L-SNMPvS

This project is a prototype implementation of the Lightweight Simple Network Management Protocol for Sensors (L-SNMPvS). The aim is to simulate a system where remote sensor devices can be monitored and managed over a network using a simplified
SNMP-like approach. The system is made up of two main parts:
- An agent, which holds and updates sensor-related data.
- A manager, which sends requests to the agent and processes its responses.
